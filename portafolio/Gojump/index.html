Plataforma de gestión de tiempos, ventas y clientes para un parque de trampolines. Sistema consistía de multiples computadoras (caja de ventas y monitoreo de tiempo) para registrar asistentes y visitas, para luego mostrar en todo el local el tiempo restante de cada asistente/visita. Cuenta con registro online para acelerar una nueva compra y una interfaz de admin para visualizar ventas asi cómo también filtrar la base de asistentes y enviar correos masivos a los correos registrados.
